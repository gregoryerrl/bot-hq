# Custom general rules

This file is **appended** to the hardcoded universal rules at session spawn — it doesn't replace them. Use it to add your own conventions, override agent defaults, or pin org-wide reminders.

Examples of things to put here:
- Org-specific commit-message conventions on top of the baked-in hygiene.
- Branch-naming conventions, release-cadence reminders.
- Cross-project lookups the index-first workflow doesn't surface.
- Reminders about tools or shortcuts that aren't in the binary's general rules.

Leave empty (or delete the file) if you have nothing to add — agents work fine on the hardcoded core alone.
